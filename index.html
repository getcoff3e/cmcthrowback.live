CMCTHROWBACK.LIVE

SMPOnline Throwback (CMCThrowback for short) is a project focusing on the restoration of CallMeCarson's Twitch Sub server SMPOnline Season 1 & 2 worlds as a completely playable survival experience with minimal differences from the original server.

The current server is in a PUBLIC TESTING phase. Things are expected to be broken, and kinks are still being worked out.
